# My Full Stack Task

A simple static website project with a home page, registration form, and shared styling.

## Files

- `index.html` - Main home page.
- `register.html` - Registration page.
- `home.html` - Alternate home or content page.
- `script.js` - JavaScript for page interactions.
- `style.css` - CSS styling for the site.

## Usage

1. Open `index.html` in a web browser.
2. Navigate to `register.html` to view the registration form.

## Notes

- This is a static website, so no server is required to run it locally.
- Edit `style.css` to customize the appearance.
- Add more pages or scripts as needed for your full stack task.
