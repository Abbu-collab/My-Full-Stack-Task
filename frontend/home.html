About SkillBridge

SkillBridge is a simple Login and Registration Portal developed to demonstrate frontend web development concepts. It allows users to create an account, log in securely, and access the home page using browser Local Storage for data management. This project is built using HTML, CSS, and JavaScript without any backend or database.
